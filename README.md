# Dusk Learning Hub

Visual-first learning resource for [Dusk Network](https://dusk.network) — a privacy-first regulated blockchain for real-world assets.

## What's Inside

- **13 sections** covering architecture, consensus, transactions, Kadcast P2P, DuskEVM, economic protocol, Dusk vs Ethereum comparison, node setup, smart contracts, wallet & keys, node maintenance, ecosystem projects, and resources
- **13+ SVG diagrams** — no ASCII art, all inline vector graphics
- **3-tab node setup** — Standard, Separated Key, and USB Cold Storage (LUKS encryption)
- **Live CoinGecko prices** for ETH and DUSK staking cost comparison
- **Ctrl+K search** across all sections
- **Ecosystem map** with 15+ community projects (NPEX, STOX, Hedger, Citadel, Chainlink, w3sper.js, etc.)

## Hosting

Single static HTML file — no build step, no dependencies. Deploy anywhere:

- **GitHub Pages**: Enable in repo Settings → Pages → Source: `main` / `root`
- **Cloudflare Pages**: Connect repo, no build command, output `/`
- **Vercel**: Import repo, framework "Other"
- **Self-host**: Any web server (Nginx, Caddy, Apache)
